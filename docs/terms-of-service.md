## 2. What StremoHub Is

StremoHub is a free, open-source media center application that:

- Provides an interface to publicly available content on YouTube

- Embeds third-party streaming sources for movies and series

StremoHub does **not** host, store, or distribute any media content. It is a client application that accesses content from third-party services.
